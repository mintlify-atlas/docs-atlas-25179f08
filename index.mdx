CUSCATLECO es el sistema de punto de venta (POS) del Restaurante Sabor Cuscatleco. Centraliza la gestión de mesas, pedidos, menú y facturación en una sola plataforma web, con paneles dedicados para cada rol operativo: Administrador, Mesero, Cajero y Cocina.

## Primeros pasos

<Steps>
  <Step title="Instala el sistema">
    Clona el repositorio, crea la base de datos MySQL con `schema.sql` e instala las dependencias con `npm install`.
  </Step>
  <Step title="Inicia el servidor">
    Ejecuta `npm start` y abre `http://localhost:3000` en tu navegador.
  </Step>
  <Step title="Inicia sesión como Administrador">
    Usa las credenciales iniciales `admin@saborcuscatleco.com` / `123` para acceder al panel de administración.
  </Step>
  <Step title="Configura el restaurante">
    Crea los empleados, asigna sus roles y verifica que las mesas estén disponibles para comenzar a operar.
  </Step>
</Steps>

<Note>
  El usuario administrador inicial viene preconfigurado en `schema.sql`. Cambia la contraseña después del primer inicio de sesión.
</Note>
